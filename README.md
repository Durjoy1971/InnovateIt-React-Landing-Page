## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/landing-page.git
   ```

2. Navigate to the project directory:

   ```bash
   cd landing-page
   ```

3. Install the dependencies:

   ```bash
   npm install
   ```
